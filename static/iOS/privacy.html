Justaway Ex

Information flowing to the notification tab when streaming can not be obtained from the Twitter API. Justaway Ex saves the streaming to the server. Justaway for iOS will get a notification content from Justaway Ex.

Justaway Ex has also been published as an open source project.

https://github.com/s-aska/Justaway-Ex

Notification tab, View Likes, function of push notifications are implemented by Justaway Ex.

Justaway Ex will be enabled When you tap the bell icon on account settings.

Justaway Ex is enabled or can be determined by the bell icon in the account settings.

If you disable the Justaway Ex will be disconnect the streaming on Justaway Ex. You can not get the notice of while you are disabled.

Contents to be stored on the server is the type and the ID and the time does not include the text. Text is obtained from the Twitter API using the ID by Justaway for iOS.

It does not provide the stored information to the outside.

Justaway Ex is using HTTPS for all communications.
